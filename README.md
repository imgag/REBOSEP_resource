# REBOSEP_resource

This repository hosts the results of our analysis of 5 different brain - meninges boundaries.  
The results are in tabular form. The columns are the genes, the standard deviation of each genes (`gene name` + `__stdev`), and the column `bin_counts` for the number of bins at each distance. The gene name columns report the average gene expression at each distance, after count normalization of each bin (target sum = 1e6).  
The rows are the distances from the tissue boundaries. The unit of measurement is one Stereo-seq bin at bin size 30.
